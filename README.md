# Skin Lesion Risk Classification (Computer Vision)

A computer vision project that classifies dermoscopic images into skin lesion categories using deep learning.

## Features
- Transfer learning with EfficientNet
- AUC, sensitivity, and confusion matrix evaluation
- Grad-CAM visual explanations
- Streamlit demo for image upload

## Dataset
HAM10000 skin lesion dataset.

## Setup
```bash
git clone https://github.com/yourusername/skin-lesion-classifier
cd skin-lesion-classifier
pip install -r requirements.txt
```
